Projects

North Carolina State University

  • Music Recommender System

    Developed a music recommender using collaborative filtering with alternating least square (ALS) algorithm. The recommender system recommends new musical artists to the user based on their listening history. Suggesting different songs or musical artists to a user is important to many music streaming services, such as Pandora and Spotify. In addition, this type of recommender system could also be used for suggesting TV shows or movies to a user on Netflix. Technology Used: Python, Apache Spark

  • Twitter Sentiment Analysis

    Processed live data streams using Apache Kafka and performed a basic sentiment analysis of real time tweets using Spark’s streaming APIs and Python.

  • Pattern Recognition and Predictive Analysis of Crime in Raleigh

    The project aimed at finding patterns in the crime incidents that occurred in Raleigh during the past few years and do predictive analysis to find the kind of crime that can happen at a specific area during a specific duration of the day. We therefore, defined a solution, that can actually predict/identify places which need more police attention at a particular duration of the time which will result in reduction of crime rate. Technologies Used: R, RStudio. GitHub Link

  • University Library Management System

    Built a Web Application for a University Library using the Ruby on Rails framework. The application implemented features for check-in, check-outs, subscriptions, email notifications etc. The project served to give us a decent exposure to the Ruby Programming language. We also gained proficiency at using the RoR framework. Technologies used: Ruby on Rails, Rspec, Heroku

  • Expertiza-Peer Review Management System

    Expertiza is a open source web application where students can submit and peer-review learning objects (articles, code, web sites, etc). Students, Instructors and admins can login, create teams, assignments and topics. We refactored a few modules of the Expertiza open source project to create a more intuitive interface to the user. Used various design principles and patterns to improve the current implementation. Technologies used: Ruby, Ruby on Rails, git

  • Automating Expertiza Setup using Chef Solo

    Expertiza is a open source web application where students can submit and peer-review learning objects like assignments, projects etc. We automated the entire process of setting up a production environment and also the deployment process for the open source project – Expertiza, by writing chef solo and knife solo recipes. Technologies used: Knife solo
